# bdk-dojo

Personal BDK Dojo learner repo.

This repo starts intentionally small. Lessons come from the public curriculum repo:

```text
https://github.com/SaucePackets/bdk-dojo-skill
```

## How to use

Ask your agent:

```text
I want to do a BDK Dojo lesson.
```

The agent should load the `bdk-dojo` skill, read the next ready scaffold, and help you implement the lesson here.

## Current state

Fresh learner repo. No lessons completed yet.

Start with:

```text
1.1 Amounts and UTXOs
```

## Safety

No real keys. No real wallet seeds. No mainnet funds.
